# Sci-Fi Reading Plans

**Live:** https://hendr15k.github.io/sci-fi-empfehlungen/

Hendrik's Sci-Fi book recommendations as an interactive reading plan tracker with:

- **143 kuratierte Titel** mit Cover-Bildern (OpenLibrary)
- **Reading Tracker** — als gelesen markieren
- **Filter & Sortierung** — nach Genres, Seiten, Rating
- **7-Tage-Leseplan** — automatic reading schedule
- **Random Picker** — wenn du nicht weißt was du als nächstes lesen sollst
- **Dark Mode** — automatisch nach System-Präferenz

Lädt Inhalte live von: https://hendr15k.github.io/sci-fi-empfehlungen/

## Tech Stack

Single HTML file with vanilla JS + CSS. No build step required.

## Development

Öffne einfach `index.html` in einem Browser.

## CI

GitHub Actions validate workflow on every push.
